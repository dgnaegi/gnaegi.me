AltpapierBot

AltpapierBot is a service that provides paper and cardboard disposal reminders for the city of Zurich via Telegram. Once set up, you will receive a reminder on the evening before and on the early morning of the disposal day for your postal code.

Setup Instructions

To set up the bot, send /postalcode 8XXX to the bot, replacing the X's with your actual postal code. To receive reminders in German, send /plz 8XXX. If you no longer need the service, you can unsubscribe from notifications by sending stop.

Additional Commands

There are some additional commands for further settings. By sending /de respectively /en, you can change your contact language. Furthermore, sometimes further notifications will be sent to you about the service. You can deactivate that by sending /stop_spam, but this comes with the chance that you're going to miss relevant information.

For more information, send /help.

Privacy Notice

Only essential information necessary to deliver disposal reminders via Telegram is stored. We collect your chat ID, which is a unique identifier for the chat session and does not reveal any personal information about you. Additionally, we store your notification preferences and your preferred language setting.

It is not possible for Telegram bots to read messages from a chat that are not explicitly addressed to them. The source code is open source and available at AltpapierBot on GitHub.
